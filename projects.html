Voice controlled Wikipedia Searcher (Name not final):

This is a project that i thought up of while browsing PYPI and seeing the Wikipedia library and Voice Recognition library. After much trouble attempting to get the voice recognition library to work, I finally was able to mess around with it and eventually develop this.

It was created in Python 3.8 and can be found here (https://github.com/Sebby37/Voice-Controlled-Wikipedia-Searcher)

Feel free to suggest any changes as it probably isnt programmed as good as it could be.

The libraries that you will need to run this are:

  • Wikipedia
  • Pyttsx3
  • SpeechRecognition
  • Requests
